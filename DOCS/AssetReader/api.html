Module assetreader

Utility for reading files from Android, since this is not possible through traditional Lua file functions.

To use the plugin, add the following in build.settings:

plugins = {
   ["plugin.AssetReader"] = { publisherId = "com.xibalbastudios" }
 }

Sample code is available here.

The Bytes type may be any object that implements ByteReader, including strings.

This plugin is designed to be called safely from other Lua processes.
