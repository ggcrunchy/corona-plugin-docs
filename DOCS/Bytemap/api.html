Module bytemap

Byte-flavored memory bitmap, useful for loading assets from byte streams (and vice versa).

To use the plugin, add the following in build.settings:

plugins = {
   ["plugin.Bytemap"] = { publisherId = "com.xibalbastudios" }
 }
