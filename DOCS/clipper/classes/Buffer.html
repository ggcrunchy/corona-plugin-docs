Class Buffer

Utility data storage type.

Methods

Buffer:Convert (path, Either)
Convert the components of a path from cInts to another numerical form, storing them in order in the buffer (the values are expected to fit within the appropriate type).

The buffer itself provides a Bytes view over these values.

Parameters:

  • path Path
  • Either string "float" or "double", describing the value type to store.
