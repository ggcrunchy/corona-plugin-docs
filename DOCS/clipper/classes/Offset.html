Class Offset

An encapsulation of the process of offsetting (inflating/deflating) both open and closed paths using a number of different join types and end types.

Offset.EndType
DOCME Examples: http://www.angusj.com/delphi/clipper/documentation/Docs/Units/ClipperLib/Types/EndType.htm

Fields:

  • ClosedPolygon Ends are joined using the JoinType value and the path filled as a polygon.
  • ClosedLine Ends are joined using the JoinType value and the path filled as a polyline.
  • OpenButt Ends are squared off with no extension.
  • OpenSquare Ends are squared off and extended delta units.
  • OpenRound Ends are rounded off and extended delta units.
