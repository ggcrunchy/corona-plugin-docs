DOCME Example: http://www.angusj.com/delphi/clipper/documentation/Docs/Units/ClipperLib/Types/PolyFillType.htm * "EvenOdd": Odd numbered sub-regions are filled, while even numbered sub-regions are not.

  • "NonZero": All non-zero sub-regions are filled.
  • "Positive": All sub-regions with winding counts > 0 are filled.
  • "Negative": All sub-regions with winding counts < 0 are filled.
