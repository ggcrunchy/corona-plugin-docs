Area (Polygon.)
This function returns the area of the supplied polygon. Depending on Orientation, this value may be positive or negative. If Orientation is true, then the area will be positive and conversely, if Orientation is false, then the area will be negative.

Parameters:

  • Polygon. Path (It's assumed that the path will be closed.)

Returns:

    number Area.

MinkowskiDiff (p1, p2[, opts])
Minkowski Difference is performed by subtracting each point in a polygon from the set of points in an open or closed path. A key feature of Minkowski Difference is that when it's applied to two polygons, the resulting polygon will contain the coordinate space origin whenever the two polygons touch or overlap. (This function is often used to determine when polygons collide.)

See here for a visual example.

Parameters:

  • p1 Path
  • p2 Path
  • opts table (mPathArray) (optional)
MinkowskiSum (P, P12[, opts])
Minkowski Addition is performed by adding each point in a polygon 'pattern' to the set of points in an open or closed path. The resulting polygon (or polygons) defines the region that the 'pattern' would pass over in moving from the beginning to the end of the 'path'.

Fields:

  • Intersection Create regions where both subject and clip polygons are filled.
  • Union Create regions where either subject or clip polygons (or both) are filled.
  • Difference Create regions where subject polygons are filled except where clip polygons are filled.
  • Xor Create regions where either subject or clip polygons are filled but not where both are filled.
