Module documentation.

Various utilities to read, write, and manipulate image data.

To use the plugin, add the following in build.settings:

plugins = {
  ["plugin.imagekit"] = { publisherId = "com.xibalbastudios" }
}

A sample project is available here (TODO).

This plugin is designed to be safely within other Lua processes.
