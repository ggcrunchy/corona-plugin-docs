Module blob

Helper to create and check for XSBlob objects.

These are writable memory buffers that can be reused across numerous calls, for instance to reduce allocations or to build up data from several writes. Blobs are readable according to the ByteReader policy. (TODO: make general reusable doc page)

XSBlob objects are not exclusive to this plugin, but this module is provided to keep it self-contained. Blobs may be shared with other plugins that consume them. (TODO: very small plugin for this purpose?)
