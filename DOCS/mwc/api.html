MakeGenerator ([opts])
Factory.

Parameters:

  • opts table

    Generator options. Fields:

    • z Z seed; if absent, uses a default.
    • w W Seed; if absent, uses a default.
    • get_zw: If true, the second return value is a getter function.

Returns:

  1. function Called as result = gen(want_real). If want_real is true, result will be a random number ∈ [0, 1); otherwise, it will be a random 32-bit integer.
  2. function or nil Called as z, w = get_zw(), to get the current generator state, which may be used to clone or restart the generator later.
